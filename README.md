## Get **DoughFin** running on your machine:

1. Clone the repository from: `git@github.com:MaryBruff/DoughFin_FE.git`
2. Move to the project directory:

```bash
cd DoughFin_FE

```

1. Grab the dependencies: `npm install`

```bash
npm install

```

1. Fire up the app: `npm start`

```bash
npm start

```
